## Описание проекта
В данной работе реализовано полноценное RESTful Web API для управления базой данных объявлений. Приложение объединяет функционал MVC (из прошлых работ) и современные методы взаимодействия через JSON.

## Реализованные требования:
1. **Web API Контроллер**: 
   - Создан `AdvertisementsApiController` с поддержкой CRUD операций (GET, POST, PUT, DELETE).
   - Реализована **серверная пагинация** для работы с большими объемами данных.
   - Вместо ID внешних ключей API возвращает смысловые названия (Категория, Автор).
2. **Документация**: 
   - Интегрирован **Swagger/OpenAPI**. Документация доступна по адресу `/swagger`.
3. **Клиентское приложение**:
   - Создана HTML-страница (`/api_client.html`), использующая JavaScript (Fetch API) для визуального выполнения всех операций.
   - Интерфейс обновляется динамически без перезагрузки всей страницы.
4. **Тестирование**:
   - Добавлены Unit-тесты для API контроллера с использованием `InMemoryDatabase`.
   - Общее количество тестов в решении: **8**.
5. **CI/CD**:
   - Настроен GitHub Actions для автоматической сборки и тестирования на платформах Windows и Linux.

## Как запустить
1. Скомпилируйте проект.
2. Перейдите по адресу `https://localhost:XXXX/api_client.html` для доступа к UI.
3. Или перейдите по адресу `/swagger` для тестирования методов API вручную.
